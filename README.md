# Redemption's Edge

A 2D top-down open-world western arcade action game — a haunted frontier outlaw tale.
Play as **Chris King ("Dckslinger")**, based out of Darryl's camp near the crooked town of
**Hicksville**. Rob, fight, explore, slip the law, and stumble toward redemption.

Built with HTML5 Canvas + vanilla JavaScript (no engine, no dependencies).

## ▶ Play

**Live:** https://soldierboy31u-spec.github.io/redemptions-edge-claude/

Or clone and open `index.html` in any modern browser (Chrome / Edge / Firefox) — no server needed.

## Controls

| Key | Action |
|---|---|
| WASD | Move |
| Mouse | Aim |
| Click / Space | Fire |
| Right-Mouse | Dead Eye (slow-mo) |
| Shift | Dash / dodge |
| Q | Throw dynamite |
| F | Lasso |
| H | Whistle your horse |
| R | Reload |
| E | Interact / mount |
| ESC | Pause |

Pick a difficulty (1 / 2 / 3) on the start screen, then press any other key to begin.

## What's in it

Weighty revolver gunplay, dash with i-frames, throwable dynamite, lasso, Dead Eye slow-mo,
a GTA-style **wanted/manhunt system** (break line of sight, lay low at camp, bribe the law),
a hand-built town with alleys and secrets, buried strongboxes, and haunted-desert landmarks
(bone arch, wagon-circle ambush, a mine you can blast open with dynamite).

*Work-in-progress prototype — feedback welcome.*
